# Smart Child Game
เกมทายสิ่งของสำหรับฝึกพัฒนาการเด็ก

### Game client
_
หน้าเกมหลัก สำหรับแสดง interface ต่างๆของเกม

**เครื่องมือที่ใช้**
- React v18.2.xx
- Node v18.17.xx
- Bootstrap 5.x.x

**ขั้นตอนการติดตั้ง**
1. Download หรือ pull code นี้ไปยังคอมพิวเตอร์ของคุณ
2. เปิด Folder ใน terminal
3. เรียกใช้คำสั่งต่อไปนี้:
    ```
    npm install
    npm run dev
    ```
